# Datablox

Datablox is a tool that can join most Roblox places via protocol links.  
It supports:  
- Joining by Place ID  
- Optional Job ID  
- Optional Private Server ID  

> ⚠️ Public version does **not** support joining users to avoid harassment.  

## Usage
1. Open the **console version** of Datablox.  
2. Enter the Place ID (and optional Job ID / Private Server ID).  
3. Hit Enter.  
4. Enjoy safe server hopping.  

## How To Get The UI Version
1. Install **LuaRT** (required — other tools won’t work).  
2. Open the Lua code in the LuaRT Editor.  
3. Run the UI version:  
   - **Option A:** Compile it with RTC  
   - **Option B:** Press F6 or F5 to run it directly  

> Note: The UI version may not compile out-of-the-box.
